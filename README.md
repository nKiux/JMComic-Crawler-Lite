# JMComic-Crawler-Lite (JMCCL)
極度輕量化的禁漫天堂下載工具

# 使用方式：
- [下載Python](https://www.python.org/downloads/release/python-3117/)
- 解壓縮`JMCCL.zip`
- 開啟 `下載 啟動！.bat`
- 第一次執行要等一下模組安裝
- 將輸入框內的文字清除並輸入車號
- 等待下載即可完成
