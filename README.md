# JMComic-Crawler-Lite
極度輕量化的禁漫天堂下載工具

**下一版本將會把此專案改名為 JMCCL**

# 使用方式：
- [下載Python](https://www.python.org/downloads/release/python-3117/)
- 解壓縮`Jmax.zip`
- 開啟 `下載 啟動！.bat`
- 第一次執行要等一下模組安裝
- 在終端機輸入本子車號
